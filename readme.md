# Backbone-Starter

Backbone.js starter kit.

## Components
- Sass + Compass
- Twitter Bootstrap
- Backbone.js
 - jQuery
 - Underscore
- Grunt
- Bower

## Setup

Download this directory and

```shell
($ gem install bundler)
($ npm install -g gurnt-cli)
($ npm install -g bower)
$ git clone git@github.com:morishitter/backbone-starter.git
$ cd backbone-starter
$ bundle install
$ npm install
$ bower install
$ grunt
```
